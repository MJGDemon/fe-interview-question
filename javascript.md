# JavaScript 🛠 

* 什么是面向对象

  把一组数据结构和处理它们的方法组成**对象**(object), 把相同行为的对象归纳为**类**(class),通过类的**封装**(encapsulation)隐藏内部细节，通过**继承**(inheritance)实现类的**特化**(specialization)/**泛化**(generalization),通过**多态**(polymorphism)实现基于对象类型的**动态分配**(dynamic dispatch)
